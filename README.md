A clean, minimal, and fast Notion-style notes application enhanced with AI.
Built using React + Vite, with smart note suggestions and improvements powered by the Google Gemini API.

Webiste hosted link -"https://wall-notes.netlify.app/"

#Features

1)Create, edit, and manage notes easily
2)Wall-grid layout for fast access
3)Gemini AI integration for enhanced note assistance
4)Quick search and clean UI
5)Uses environment variables for secure key handling
6)Super-fast development with Vite

#Tech Stack
React.js, Vite, JavaScript (ES6+), CSS / HTML, Google Gemini API, Node.js & npm


#How to run locally
1)Install dependencies -npm install

2) Add your Gemini API key - Create .env.local and add:
                              GEMINI_API_KEY=your_api_key_here
                              
3)Start the development server -npm run dev
